Software Distribution Page
Bioinformatics and computational biology software and databases

The bioinformatics and computational biology software and databases offered through this page were created by researchers at the National Institute of Allergy and Infectious Diseases (NIAID) which is part of the US National Institutes of Health (NIH).


SPICE

SPICE is a data mining software application that analyzes large FLOWJO data sets from polychromatic flow cytometry and organizes the normalized data graphically. SPICE enables users to discover potential correlations in their experimental data within complex data sets.

Go to application

PathogenAR

PathogenAR is a free smartphone app (available in iOS and Google Play stores) that provides an interactive, augmented reality (AR) experience for learning about infectious pathogens and biomolecules.

Go to application

Cytometry Template Editor

Cytometry Template Editor is a Window executable zip file that will allow you to edit Cytometry templates.

Go to application

Retired Applications

The following applications were previously developed by NIAID but are no longer maintained or supported. They are listed here for historical reference only.

  • Sage (Retired)

    Sage was a comprehensive bioinformatics analysis platform that provided integrated tools for genomic data processing and visualization.

  • Additivity (Retired)

    Additivity was a web-based tool for analyzing drug combination effects and determining synergistic, additive, or antagonistic interactions between compounds in biological systems.

  • MAPCD (Retired)

    MAPCD (Mapping and Analysis Platform for Cell-type Deconvolution) was a comprehensive tool for analyzing cellular composition in bulk RNA-seq data and performing cell-type deconvolution analysis.

  • SPADE (Retired)

    SPADE (Spanning-tree Progression Analysis of Density-normalized Events) was a data analysis algorithm for identifying cellular populations in high-dimensional flow cytometry data.

  • JoinSolver (Retired)

    JoinSolver was a computational tool for analyzing and solving complex biological network problems, providing insights into molecular interactions and pathway analysis.
